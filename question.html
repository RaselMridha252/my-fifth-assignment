Question-1: What are the different ways to select an element in the DOM?

Ans : getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll().


Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: innerHTML html ট্যাগ সহ সম্পূর্ণ কন্টেন্ট প্রদান করে। innerText শুধুমাত্র টেক্সট প্রদান করে। textContent html এর ভিতরে না থাকলেও টেক্সট প্রদান করতে পরে।


Question-3: What is event delegation in the DOM?

Ans: DOM দিয়ে আমরা html ও css নিয়ন্ত্রণ ও পরিবর্ন করতে পারি।


Question-4: What is event bubbling in the DOM?

Ans: event bubbling দিয়ে তার আগের ট্যাগএর মধ্যে প্রবেশ ও পরির্তন করে।


Question-5: How do you create, add, and remove elements using JavaScript?

Ans: JavaScript দিয়ে function এর মাধ্যমে এগুলো করতে হবে।
